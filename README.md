## 🚀 Production Artifacts

### 🧠 DocChat | *On-Device AI Pipeline*
> **Flutter • Dart • Whisper • Gemma 2B • Silero • Local Storage**

A privacy-first, fully offline AI assistant. The entire pipeline—speech recognition (Whisper), language model (Gemma 2B), and text-to-speech (Silero)—runs on-device with zero internet required. Engineered an offline-first sync queue that correctly merges state without conflicts.

### 🏛️ Deliberate | *Real-Time Debate Graph Engine*
> **Flutter • Node.js • Express • MongoDB • Socket.io**

A structured debate platform featuring a real-time logical fallacy detector (ad hominem, straw man, etc.) and a dynamic cooldown system. Built a custom quality scoring engine weighing evidence (30%), logic (25%), peer review (20%), engagement, and novelty, visualizing the entire debate tree.

### 🔍 DocMind | *Enterprise Document Intelligence*
> **React.js • Node.js • Express • MongoDB • LLM APIs • WebSockets**

A distributed team document intelligence platform. Implemented hybrid semantic + keyword search with multi-document reasoning. Authored a smart caching layer that reduced AI API costs by 68% while enabling live WebSocket collaboration across teams.

### 📉 Depression Risk Prediction | *Recall-Optimized ML*
> **Python • XGBoost • SMOTE • SHAP • Pandas**

Engineered a recall-optimized prediction model on 27,901 records, focusing strictly on minimizing false negatives. Handled severe class imbalance using SMOTE, implemented SHAP for model explainability, and conducted a rigorously audited gender-based fairness evaluation.
